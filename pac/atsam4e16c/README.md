# Microchip/Atmel ATSAM4e16c PAC for Rust

This repository holds the Peripheral Access Crate for the Atmel `ATSAM4e16c` microcontroller.

[![Build Status](https://travis-ci.org/atsam4-rs/atsam4e16c-pac.svg?branch=master)](https://travis-ci.org/atsam4-rs/atsam4e16c-pac)

This **P**eripheral **A**ccess **C**rates (**PAC**s) is automatically generated, and provides low-level access to the peripherals specified by a device's SVD file.
